Java-mirror-api-emulator

We redirect the google java mirror-api to the Scarygami/mirror-api emulator. The purpose is to help java developers and non-glass-explorers to get an idea of what will be possible.

Quick start documentation

Install your own Google Glass emulator

We will use the Scarygami Google Glass emulator, he has done the main work. He is the one who makes all possible. You need to have installed and deployed your own Scarygami/mirror-api emulator.See this document for a detailed description of what have been done and how it works.

Our goal is to send orders from java to this emulator.

Get the official Google Glass/mirror-quickstart-java

We will use the googleglass/mirror-quickstart-java as the quickstart project. You will have to clone the project and import in your favorite IDE.

In the oauth.properties you will set your own Google Glass emulator credentials.

Setup the java-mirror-emulator

Okey, the most difficult part is done. The only remaining thing is to configure the official Google Glass/mirror-quickstart-java to use your emulator. This is where this project magic starts. Operations to perform :

  1. Download the last version of Java-mirror-api-emulator.jar

  2. Modify the pom.xml to use the mirror Java-mirror-api-emulator.

    3. <dependency>
      <groupId>com.google.apis</groupId>
      <artifactId>google-api-services-mirror-emulated</artifactId>
      <version>v1-emulated</version>
</dependency>

  3. Modify the web.xml for images attachement support. Add the following lines : 

    4. <servlet>
    <servlet-name>DownloadAttachementsServlet</servlet-name>
    <servlet-class>com.google.api.services.mirror.utils.DownloadAttachementsServlet</servlet-class>
</servlet>
<servlet-mapping>
    <servlet-name>DownloadAttachementsServlet</servlet-name>
    <url-pattern>/attachements/*</url-pattern>
</servlet-mapping>

  4. Configure the quickstart to use your emulator URL as root URL of the service. Assuming that your emulator URL is https://<your_app_id>.appspot.com. Change the way to retrieve the Mirror in the com.google.glassware.MirrorClient class. Use the com.google.api.services.mirror.utils.EmulatorUtil to have something like this :

    5.   
  public static Mirror getMirror(Credential credential) {
    return EmulatorUtil.getMirror(credential, "PG Java Starter", 
    "https://<your_app_id>.appspot.com");
  }

Launch your GlassWare

Okey your Glassware is the quickstart, but we love this term. The only remaining thing is to get your glassware project, and execute comand :

mvn jetty:run.

In your favorite browser , open the URL :

localhost:8080
