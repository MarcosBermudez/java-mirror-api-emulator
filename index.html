Java-mirror-api-emulator

We redirect the google java mirror-api to the Sacygami/mirror-api emulator. The purpose is to help java developers and non-glass-explorers to get an idea of what will be possible.

Quick start documentation

Install you own Google Glass emulator

We use the Scarygami Google Glass emulator, he has done the main work. He is the one who makes all posible. You need to have installed and deployed your own Sarigami/mirror-api emulator.See this document for a detailed description of what this does and how it works. We will send orders from java to this emulator.

Get the official Google Glass/mirror-quickstart-java

We will use the googleglass/mirror-quickstart-java as the quickstart project. You will have to clone the project and import in your favorite IDE. Is important to configure security, folow this guide to enable the oauth secuirity.

In the oauth.properties you will set your own Google Glass emulator credentials.

Setup the java-mirror-emulator

Okey, the most dificult part is done. The only remaining thing is to configure the official Google Glass/mirror-quickstart-java to use your emulator. This is where this project magic starts. Operaions to perform:

  1. Download the last version of Java-mirror-api-emulator.jar

  2. Modify the pom.xml to use the mirror Java-mirror-api-emulator

  3. Configure your emulator URL

Launch your GlassWare

Okey your Glassware is the quickstart, but I love this therm. The only remaining thing is to get to the folder of the quickstart, and execute comand :

mvn jetty:run.

Launch your browser open the URL:

localhost:8080
